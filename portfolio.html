Solutionshub

A website where Ironhack students can browse, upvote and serach through solutions to exercises given during the bootcamp. At the Ironhack bootcamp students submit their exricises by doing a pull request of the starter code of that exercise. All solutions is there for on GitHub, but it is hard to find the best one. This website solves that problem.

Technologies Used

  • Node
  • MongoDB
  • Mongoose
  • Express
  • React
  • Reactstrap
  • Passport-github
  • GitHub API
